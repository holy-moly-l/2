# Голосовой ассистент для Arch Linux

Голосовой ассистент на Python, который распознает русскую речь и выполняет команды в системе Arch Linux.

## Возможности

Ассистент умеет выполнять следующие команды:

- **"открой браузер"** - открывает браузер (Chrome, Firefox, Chromium, Brave)
- **"открой терминал"** - открывает терминал
- **"открой файловый менеджер"** - открывает файловый менеджер
- **"обновить систему"** - обновляет систему через pacman
- **"покажи дату"** - показывает текущую дату
- **"покажи время"** - показывает текущее время
- **"выключи компьютер"** - выключает компьютер
- **"перезагрузи компьютер"** - перезагружает компьютер
- **"сделай скриншот"** - делает скриншот экрана
- **"блокируй экран"** - блокирует экран
- **"открой калькулятор"** - открывает калькулятор
- **"открой настройки"** - открывает настройки системы
- **"помощь"** - показывает список всех команд
- **"стоп"** или **"выход"** - останавливает ассистента

## Установка зависимостей

### 1. Установите системные пакеты:

```bash
sudo pacman -S python-pip python-pyaudio portaudio
```

### 2. Установите Python-библиотеки:

```bash
pip install SpeechRecognition
```

### 3. (Опционально) Для лучшего распознавания речи:

```bash
pip install pocketsphinx
```

## Запуск

```bash
python3 voice_assistant.py
```

Или сделайте файл исполняемым:

```bash
chmod +x voice_assistant.py
./voice_assistant.py
```

## Использование

1. Запустите ассистента
2. Дождитесь сообщения "Слушаю..."
3. Произнесите команду голосом
4. Ассистент выполнит команду

## Примечания

- Для работы требуется микрофон
- Распознавание речи использует Google Speech Recognition API (требуется подключение к интернету)
- Некоторые команды могут требовать прав суперпользователя (например, обновление системы, выключение)
- Ассистент автоматически определяет установленные в системе приложения

## Добавление своих команд

Вы можете добавить свои команды, отредактировав словарь `self.commands` в классе `VoiceAssistant`:

```python
self.commands = {
    "ваша команда": self.ваш_метод,
    # ...
}
```

Затем добавьте метод для выполнения команды:

```python
def ваш_метод(self):
    """Описание команды"""
    # Ваш код
    subprocess.run(["ваша", "команда"])
```

## Требования

- Python 3.6+
- Arch Linux (или другой дистрибутив с pacman)
- Микрофон
- Подключение к интернету (для распознавания речи)
